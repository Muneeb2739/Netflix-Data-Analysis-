# 🎬 Netflix Data Analysis Project

## 📌 Overview

This project analyzes Netflix movies and TV shows data to uncover insights about content distribution, genres, and country-wise production trends.
The analysis is performed using **Python, Pandas, and Data Visualization libraries**.

---

## 📂 Dataset

* Dataset: *Netflix Movies and TV Shows*
* Source: Kaggle

The dataset contains information such as:

* Type (Movie / TV Show)
* Title
* Country
* Genre 
* Release Year
* Date Added
* Rating

---

## 🎯 Objectives

The project aims to answer the following questions:

* What is the ratio of Movies vs TV Shows?
* Which genre is most common on Netflix?
* Which country produces the most content?
* How has content growth changed over time?

---

## 🛠️ Technologies Used

* Python 
* Pandas
* Matplotlib
* Seaborn
* Google Colab 

---

## 🔍 Data Cleaning

* Handled missing values in `country`, `director`, and `cast`
* Removed rows with critical missing data
* Converted `date_added` to datetime format

---

## 📊 Key Analysis & Visualizations

### 1️⃣ Movies vs TV Shows

* Bar chart comparison
* Insight: Movies are more common than TV Shows

---

### 2️⃣ Most Common Genres

* Top 10 genres using bar chart
* Insight: Drama and International Movies dominate

---

### 3️⃣ Top Content Producing Countries

* Country-wise analysis
* Insight: United States produces the most content

---

### 4️⃣ Content Growth Over Time

* Line chart based on year added
* Insight: Rapid growth after 2015

---

## 📈 Key Insights

* Netflix has significantly more **Movies than TV Shows**
* **Drama** is the most popular genre
* **USA** is the leading content producer
* Content addition increased rapidly in recent years

---

## 🙌 Conclusion

This project demonstrates how data analysis can uncover meaningful insights from real-world datasets and helps in understanding content trends on streaming platforms like Netflix.
